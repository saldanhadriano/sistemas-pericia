📝 Resumo do Sistema Criado
Nome: Sistema de Gestão de Perícias
Tecnologias: Python + Streamlit + SQLite
Localização: Desktop/sistema-pericias/
Arquivos importantes:

app.py - Código principal
pericias.db - Banco de dados (backup obrigatório!)

Para executar:
bashcd Desktop/sistema-pericias
streamlit run app.py
Funcionalidades:

✅ Cadastro de perícias (Vara, Processo, Classe, Datas, Valores)
✅ Gestão de Status (Aberto, Em Revisão, Entregue, Recebida)
✅ Entrevistas com status (Pendente/Realizada)
✅ Calendário de próximas entrevistas
✅ Resumo financeiro
✅ Cálculo automático de prazo restante
✅ Todas as datas em formato DD/MM/YYYY
